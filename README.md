# Clawfund Agent

Agent tooling for Solana and memecoin markets. This repo is the documentation and skills hub for the Clawfund system.

---

## What This Repo Is

- A clean, readable explanation of the system
- A skills-first foundation for agent builders
- A place to ship examples and references

---

## Start Here

- `docs/quickstart.md` — fastest path to a working mental model
- `docs/architecture.md` — system map and data flow
- `docs/cycle.md` — 5-minute cycle mechanics
- `docs/verification.md` — wallet and social verification
- `docs/skills.md` — skills philosophy and references

---

## Included Assets

- `skills/openclaw-solana/` — OpenClaw Solana skill (vendored)
- `examples/basic-agent.ts` — minimal agent example
- `docs/` — full documentation set

## Mission

Clawfund began as a 5-minute agentic fund experiment. The broader mission is larger:
**build a practical agent tool system for Solana and meme-coin markets.**

## Safety & Scope

This repo is educational and infrastructure-focused. It is **not** financial advice.
Run on devnet or with strict safeguards before touching mainnet capital.
